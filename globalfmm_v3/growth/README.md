# Growth & Marketing Automation

This directory contains Python scripts and automation tools for **Line 3 (Growth)** of the Advance ecosystem. These scripts are intended to be run locally or via CI/CD pipelines, NOT deployed to the public web server.

## Contents

- `seo_audit.py`: Script to analyze internal linking and keyword presence.
- `linkedin_agent.py`: (Planned) Agent to monitor and suggest LinkedIn interactions.
- `kpi_report.py`: (Planned) Script to aggregate growth metrics.

## Setup

1. Create a virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
